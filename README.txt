INSTALLATION
1. Upload zip folder in your root directory and unzip.
2. Import djunehor.sql into database using phpmyadmin. New or existing database
3. Open includes/config.php file and edit database details.
4. If you changed table names, edit as appropriate in config.php
5. Default admin folder is "test-admin". If you wish to change admin folder url, change the admin folder name to anything and edit as appropriate in the config.php file
6. Access user login page with yourwebsite.com/user/userLogin
7. Access admin login page with yourwebsite.com/admin-folder-name/adminLogin
